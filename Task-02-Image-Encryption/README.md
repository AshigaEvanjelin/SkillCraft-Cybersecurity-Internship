# Task 02 - Image Encryption Tool

## Objective
Develop a simple image encryption and decryption tool using pixel manipulation techniques.

## Description
This project encrypts and decrypts images by modifying RGB pixel values using a user-defined key. The encryption process changes the pixel values, making the image unreadable, while the decryption process restores the original image using the same key.

## Features
- Image Encryption
- Image Decryption
- User-defined Encryption Key
- File Selection Dialog
- Menu-driven Interface
- Input Validation
- Continuous Execution Until Exit

## Technologies Used
- Python
- Pillow (PIL)
- Tkinter
