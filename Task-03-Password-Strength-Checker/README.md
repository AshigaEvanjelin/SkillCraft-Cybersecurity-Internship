# Task 03 - Password Strength Checker

## Objective

Build a tool that assesses the strength of a password based on criteria such as length, presence of uppercase and lowercase letters, numbers, and special characters.

## Features

* Password Strength Analysis
* Length Validation
* Uppercase Letter Detection
* Lowercase Letter Detection
* Number Detection
* Special Character Detection
* Menu-Driven Interface
* Continuous Execution Until Exit

## Technologies Used

* Python

## How to Run

1. Open the project folder in VS Code.
2. Run the program:

python password_checker.py

3. Enter a password to check its strength.

## Strength Criteria

* Weak: Satisfies 1–2 conditions
* Medium: Satisfies 3–4 conditions
* Strong: Satisfies all 5 conditions

## Outcome

Successfully developed a Password Strength Checker that evaluates password security based on multiple criteria and classifies passwords as Weak, Medium, or Strong.
