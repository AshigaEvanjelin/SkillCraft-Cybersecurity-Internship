# Task 04 - Keyboard Event Monitor

## Objective

Develop a keyboard event monitoring application that captures and logs keys entered within the application's own window.

## Features

* Real-time key event detection
* Key logging within the application window only
* Save logged keys to a text file
* User-controlled start and stop
* Simple graphical interface

## Technologies Used

* Python
* Tkinter

## Outcome

Successfully implemented a keyboard event monitoring application that demonstrates keyboard event handling and file logging while operating only within the application's own interface.
